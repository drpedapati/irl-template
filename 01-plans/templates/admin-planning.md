# IRL Plan: Administrative Documents

## Structure
Create on first run:
- drafts/
- final/
- logs/

## Pre-instruction Hooks
- git commit before edits
- do not edit this file

## Strategy
- Document-first: clear, versioned artifacts
- Naming: YYMMDD-document-name.md
- Markdown for drafts, PDF/Word for distribution
- Keep documents concise and actionable

## Document Types
- **Plans**: strategic plans, project plans, roadmaps
- **Reports**: status updates, summaries, assessments
- **Memos**: internal comms, decision records
- **Proposals**: funding requests, policy changes
