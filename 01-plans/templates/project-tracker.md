## Project Metadata
- Project: [Name]

## Approach
- Single source of truth for tasks and schedule
- Tasks in tasks/tasks.csv
- Timeline in tasks/schedule.md
- Weekly review cycle

## Task Schema (tasks.csv)
| Column | Values |
|--------|--------|
| id | P001, P002... |
| task | description |
| owner | person |
| status | not_started / in_progress / blocked / done |
| priority | high / medium / low |
| due | YYYY-MM-DD |
| depends_on | task ids |
| notes | context |
